## ⚡ Quick Start Commands

### 1. Install Dependencies
```bash
cd Bias_Mitra
flutter pub get
```

### 2. Configure Firebase (IMPORTANT!)
```bash
# Install FlutterFire CLI
dart pub global activate flutterfire_cli

# Run configuration
flutterfire configure --project=bias-mitra-solution-challenge
```

When prompted, select:
- ✅ Android
- ✅ iOS
- ✅ Web
- ✅ Windows (optional)
- ✅ macOS (optional)

### 3. Run the App
```bash
# Web (Chrome)
flutter run -d chrome

# Android
flutter run -d android

# iOS
flutter run -d ios
```

## 🚨 Common Issues

### Issue: "Cannot find flutter"
```bash
# Add Flutter to PATH
export PATH="$PATH:$FLUTTER_PATH/bin"
```

### Issue: "firebase_options.dart not found after flutterfire configure"
```bash
# Clean and regenerate
flutter clean
flutterfire configure --project=bias-mitra-solution-challenge
```

### Issue: "Google Sign-In fails on Web"
Visit Firebase Console Settings → OAuth Redirect URIs → Add `http://localhost:7357`

### Issue: Android build fails
```bash
flutter clean
flutter pub get
flutter run -d android
```
